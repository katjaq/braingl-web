# Example for online publications with braingl-web including interactive 2D figures linked with the interactive 3D viewer

## Katja Heuer, Ralph Schurade, Angela D Friederici & Alfred Anwander



Max Planck Institute for Human Cognitive and Brain Sciences, Leipzig, Germany
This is an example for an online publication with interactive 3D figures. The code of the template can be found as example-pub at github.com/rschurade/braingl-web.In 
addition to such an online article template, braingl-web also provides a framework to create an interactive 3D website for your data which you can easily link with your journal 
publication. The code for the template can be found as 'brainnetworks' within braingl-web.This image is a collage as to what the online publication will look like once the interlinked figure and viewer will be implemented.
